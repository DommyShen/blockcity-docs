# 开发流程

**1. 注册**

在布洛克城开发者平台注册账号，并按照指引完善账号信息；

布洛克城开发者平台网址：
[https://developer.gxb.io/](https://developer.gxb.io/)

**2. 创建应用**

在应用管理页"创建应用"，填写应用名称、应用简介、应用图标等信息等；

**3. 开发小应用**

开发者可下载开发者工具、参考开发文档进行小应用的开发和调试；

**4. 提交审核和发布**

完成开发后，提交至布洛克城团队审核，审核后即可发布。
